# VIRIDYAN WEBSITE
A Virtual Medicinal Garden for Learning and Exploration.

## Project Description
Viridyan is an interactive 3D virtual herbal garden where users can explore medicinal plants, view plant information, and learn about their health benefits.

The system integrates:

- React.js frontend
- Three.js 3D environment
- Node.js backend
- MongoDB database

Users can navigate inside the garden and click plants to see their medicinal properties.

---

## Features
- 🌿 Interactive 3D herbal garden
- 🌱 Real medicinal plant models
- 📖 Plant information panel
- 🔍 Search plants by disease
- 🧭 Garden navigation
- 📚 AYUSH medicinal system data

## 3D Models Setup

3D plant models should be placed in:
frontend/public/models/plants/

Example models:
tulsi.glb
neem.glb
giloy.glb
aloe.glb
amla.glb


These models are not included in the GitHub repository because of GitHub file size limits.

## Installation

### Backend
cd backend
npm install
node server.js

Server runs on:
http://localhost:5000


---

### Frontend
cd frontend
npm install
npm start

Frontend runs on:
http://localhost:3000
